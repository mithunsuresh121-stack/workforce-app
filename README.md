# workforce-app
Workforce App is a complete platform for managing employee attendance, shifts, and leave. It supports role-based permissions, clock-in/out, break tracking, and admin overrides. The app includes a web interface, a mobile app, and a backend API, making it easy for teams to manage workforce operations efficiently.
